### 🧪 Featured Projects

#### 🤖 Intelligent Physics Teaching Assistant
> An LLM-powered system for university physics courses, supporting Q&A, automated grading, and learning diagnosis.
- Built a multi-stage RAG pipeline, reducing model hallucination from 35% to 10%.
- Implemented step-by-step grading with event sequence diagrams, achieving 92% accuracy on full-score assignments and a QWK of 0.89.

- Tech stack: Python, LangChain, Neo4j, MySQL, FastAPI
- Repository: http://tjzhwl.com.cn/
